# 🍞 Panificadora e Confeitaria Tia Bia

Projeto desenvolvido para a **Panificadora e Confeitaria Tia Bia**, com foco em **catálogo digital, montagem de pedidos e envio direto para o WhatsApp**.

O sistema foi pensado para oferecer uma experiência simples, rápida e responsiva, facilitando o atendimento de clientes em **desktop, tablet e celular**.

---

## 🚀 Funcionalidades

* 📋 Catálogo organizado por categorias

  * Salgados
  * Doces
* 🖼️ Cards com imagem, nome, preço e variações
* ➕ Adição de produtos ao pedido
* 📱 Layout totalmente responsivo
* 📅 Seleção de data e horário
* 🚫 Bloqueio de pedidos aos domingos
* ⏰ Validação de horário de atendimento

  * Atendimento: **06:00 às 19:30**
* 🚚 Opção de retirada ou entrega
* 📍 Campo de endereço dinâmico

  * Exibe apenas quando **Entrega** é selecionado
* 💳 Forma de pagamento
* 💰 Cálculo automático do valor total
* 📲 Envio do pedido formatado para WhatsApp
* 🔔 Notificação visual ao adicionar itens
* 🍰 Aviso para encomenda de tortas

---

## 🛠️ Tecnologias utilizadas

* **HTML5** → estrutura do site
* **CSS3** → estilização e responsividade
* **JavaScript** → lógica do pedido, cálculos e interações

## 🎯 Objetivo do projeto

O principal objetivo foi criar um **sistema real para uso comercial**, permitindo que clientes façam pedidos de forma prática pelo celular e enviem diretamente para o WhatsApp da empresa.

Além disso, o projeto foi desenvolvido para fortalecer habilidades em:

* Responsividade
* UX/UI
* Manipulação do DOM
* Eventos JavaScript
* Integração com WhatsApp
* Boas práticas de front-end
